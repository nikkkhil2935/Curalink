# Curalink Design System

## Visual Direction
- Domain: medical research intelligence UI
- Style: dark, data-dense, high legibility
- Brand mood: technical, trustworthy, evidence-first

## Tokens
- Background: `#070a12`, `#101726`
- Surfaces: `#131d2d`, `#1a263a`
- Border: `#24324a`
- Text primary: `#f3f7ff`
- Text secondary: `#c6d3eb`
- Text muted: `#8ea1c2`
- Brand: `#1d4ed8`, `#2563eb`
- Success: `#16a34a`
- Warning: `#ca8a04`
- Error: `#dc2626`

## Typography
- Base font family: `'Segoe UI', Tahoma, Geneva, Verdana, sans-serif`
- Headings: bold to black weights
- Body text: slate-toned for contrast on dark surfaces

## Components
- Cards: rounded-xl to rounded-2xl with subtle borders
- Primary actions: blue filled buttons
- Secondary actions: outlined slate buttons
- Status pills: source-specific and evidence-specific color coding
- Tabs: compact uppercase labels with active/inactive visual contrast

## Motion
- Subtle transitions for hover/focus and loading feedback
- Avoid heavy motion; prioritize clarity over novelty

## Layout
- Desktop: multi-panel split layouts for research workflow
- Mobile: single-focus pane navigation with quick switching
- Spacing rhythm: 4px scale, 12-24px core block spacing
